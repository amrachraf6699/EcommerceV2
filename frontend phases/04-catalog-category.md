# Phase 4: Catalog and Category Listing

## Goal
Turn the storefront into a browsable shopping experience with strong listing UX on both desktop and mobile.

## Scope
- Build category listing pages.
- Build catalog/product listing pages.
- Add filter and sort UI.
- Add mobile filter drawer behavior if needed.
- Build the storefront product card system.
- Ensure listing grids do not collapse awkwardly on small screens.

## Deliverables
- Category and product archive routes.
- Product cards with the intended hover treatment.
- Filtering and sorting UI that fits RTL layout.

## Test Checklist
- Catalog pages render real products and categories.
- Filter and sort controls update results correctly.
- Mobile filter UI opens and closes cleanly.
- Product cards remain readable and visually balanced on narrow screens.

## Acceptance
- Browsing products is practical on mobile and desktop.
- Cards feel premium and interactive.
- Listing structure is ready for deep product exploration.
